## Highlight

- Core roff/mdoc parser producing structured JSON.
- Markdown output with YAML front matter.
- Progressive disclosure `view` command for AI-friendly man page inspection.
- `bench` command for parser performance testing against local man pages.
- Multi-target release pipeline with cosign signed artifacts.
- OpenSSF Scorecard and CodeQL tracking enabled.

## Add

- `roff tojson` and `roff tomd` commands.
- `roff view` with `--description`, `--synopsis`, `--options`, `--see-also`, `--meta`, `--outline`, etc.
- `roff bench` with `--count` and `--all`.
- Library API: `parse_to_json`, `parse_to_string`, `to_markdown`, `ViewOptions`, `view`.

## Note

- Release assets are signed with cosign `.sigstore.json` bundles.
